Detailed Description

Geometry buffers holder.

Author
Sylvain DOREMUS
Version
0.7.0.0
Date
18/11/2012
Remarks
Allows implementations to use API specific optimisations (like OpenGL Vertex array objects)
Acquires it's buffers responsibility when constructed
