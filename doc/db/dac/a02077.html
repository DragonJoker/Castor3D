Detailed Description

SSAA Render technique class.

Author
Sylvain DOREMUS
Version
0.7.0.0
Date
12/11/2012
Remarks
It creates a supersampled frame buffer and needed attachments,
it renders to the singlesampled frame buffer
then it blits this singlesampled frame buffer to the main one with appropriate resize
