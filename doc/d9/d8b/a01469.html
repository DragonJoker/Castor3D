Detailed Description

Deferred lighting Render technique class.

Author
Sylvain DOREMUS
Version
0.7.0.0
Date
12/11/2012
Remarks
Creates up to 6 render buffers (position, diffuse, normals, tangents, bitangents, specular)
it renders to these buffers (Geometry Pass)
then renders these buffers into the final one, adding the lights (Light Pass)
