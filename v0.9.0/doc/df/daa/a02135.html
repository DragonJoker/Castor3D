Detailed Description

Post render effect base class.

Author
Sylvain DOREMUS
Version
0.7.0.0
Date
20/11/2012
Remarks
A post render effect is an effect applied after 3D rendering and before 2D rendering.
Post render effects are applied in a cumulative way.
